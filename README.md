Local desktop helper for Git and Jira ticket workflows
- Build a single command to commit (with message) and/or create a pull request, with buttons to add the ticket URL and QA environment URL to the PR description
- Generate an embedded Teams link that includes the ticket key and title
- Generate easy-to-spot stash commands with a timestamp, plus an optional short suffix

## Run
- First time: `npm install`
- Then (and next times): `npm run dev`
